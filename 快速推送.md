# 快速推送到 GitHub

## 最简单的方法（推荐）

### 步骤 1：创建 Personal Access Token

1. 访问：https://github.com/settings/tokens
2. 点击 **"Generate new token"** → **"Generate new token (classic)"**
3. 输入名称：`CLARITY Upload`
4. 选择过期时间（建议 90 天或自定义）
5. **勾选 `repo` 权限**（完整仓库权限）
6. 点击 **"Generate token"**
7. **立即复制 token**（只显示一次！）

### 步骤 2：推送代码

在终端运行：

```bash
cd "/Users/xiting/Desktop/nus study/semester2/CDE5301/job-seeker-coding"
git push -u origin main
```

当提示时：
- **Username**: 输入 `xt0103`
- **Password**: **粘贴你的 Personal Access Token**（不是 GitHub 密码）

## 或者使用脚本

运行我创建的辅助脚本：

```bash
cd "/Users/xiting/Desktop/nus study/semester2/CDE5301/job-seeker-coding"
./push.sh
```

脚本会引导你完成推送过程。

## 验证

推送成功后，访问 https://github.com/xt0103/CLARITY 应该能看到所有文件。

## 如果遇到问题

- **认证失败**：确保使用的是 Personal Access Token，不是 GitHub 密码
- **权限错误**：确保 token 有 `repo` 权限
- **网络问题**：检查网络连接
