# next-web (CLARITY Job Seeker UI)

## Requirements
- Node.js 18+ (or 20+)

## Local setup

```bash
cd apps/next-web
npm install
```

Create `.env.local` (see `env/ENV_TEMPLATES.md`, or copy from `apps/next-web/env.example`):

```bash
NEXT_PUBLIC_API_BASE_URL=http://localhost:8000
```

Run:

```bash
npm run dev
```
