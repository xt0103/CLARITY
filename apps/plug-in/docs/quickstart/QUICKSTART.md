# 快速开始指南

## 安装步骤

1. 打开Chrome浏览器
2. 访问 `chrome://extensions/`
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择本项目目录

## 使用流程

### 第一步：准备CSV文件

创建一个CSV文件，或使用项目中的 `examples/导入个人信息.csv` 作为示例：

```csv
姓名,邮箱,电话,地址,公司
张三,zhangsan@example.com,13800138000,北京市朝阳区,ABC公司
李四,lisi@example.com,13900139000,上海市浦东新区,XYZ公司
```

### 第二步：导入数据

1. 点击浏览器工具栏中的插件图标
2. 点击"⚙️ 打开设置"
3. 点击"选择CSV文件"
4. 选择您的CSV文件
5. 查看数据预览确认导入成功

### 第三步：检测表单字段

1. 打开需要填写表单的网页（例如：招聘网站、注册页面等）
2. 在设置页面中，点击"检测当前页面表单字段"
3. 查看检测到的字段列表

### 第四步：配置字段映射

1. 点击"自动匹配"按钮（插件会自动匹配字段）
2. 检查每个字段的映射是否正确
3. 手动调整不正确的映射
4. 点击"保存配置"

### 第五步：一键填写

1. 在需要填写表单的网页上，点击插件图标
2. 确认"表单字段"和"数据状态"都显示正常
3. 点击"一键填写"按钮
4. 检查表单是否已正确填写

## 常见问题

**Q: 为什么检测不到表单字段？**
A: 确保页面已完全加载，某些动态加载的表单可能需要等待几秒后再检测。

**Q: 自动匹配不准确怎么办？**
A: 可以在设置页面中手动调整每个字段的映射关系。

**Q: 填写后表单没有反应？**
A: 某些网站可能需要手动触发验证，填写后可以尝试点击提交按钮或切换焦点。

**Q: 支持哪些字段类型？**
A: 支持文本输入框、下拉选择、多行文本、复选框、单选框。不支持文件上传。

## 提示

- 建议先在一个测试页面上试用，熟悉流程后再在实际场景中使用
- 如果CSV数据有更新，可以重新导入CSV文件
- 字段映射配置会保存，下次使用时无需重新配置
